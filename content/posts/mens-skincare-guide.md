## 🧴 メンズ必須アイテム5選

### ① 洗顔料（最重要）
**朝晩1日2回**、ぬるま湯＋しっかり泡立てで使用。ゴシゴシ擦らないのがコツ。

### ② 化粧水
洗顔後30秒以内に使用。**500円玉サイズ**を手に取り、顔全体に馴染ませる。

### ③ 乳液/クリーム
化粧水の水分を逃さない**フタの役割**。少量でOK。

### ④ 日焼け止め
紫外線は**シミ・シワの最大原因**。SPF30以上を毎朝塗布。

### ⑤ 美容液（余裕があれば）
ニキビ跡・シミ対策にビタミンC配合のものを。

## 🎯 肌悩み別・おすすめの選び方

{{< point title="肌タイプ別おすすめ" >}}
- **テカリ・脂性肌** → さっぱり系化粧水 + ジェル乳液
- **乾燥・カサつき** → しっとり系化粧水 + クリーム
- **ニキビが多い** → 薬用・ノンコメドジェニック表記のもの
- **敏感肌** → 無香料・アルコールフリー
{{< /point >}}

---

## ⏰ 3分でできるメンズスキンケアルーティン

```
【朝】洗顔（30秒）→ 化粧水（30秒）→ 乳液（30秒）→ 日焼け止め（30秒）
【夜】洗顔（60秒）→ 化粧水（30秒）→ 乳液（30秒）
```

合計**3分程度**で完了。毎日続けることが何より大切です。

## ❓ よくある質問

{{< accordion title="ユニセックス（女性用）の化粧水を使ってもOK？" >}}
もちろんOKです。ただし男性の肌は**皮脂量が女性の2〜3倍**なので、さっぱり系のメンズ専用ブランドの方が合いやすいです。
{{< /accordion >}}

{{< accordion title="ドラッグストアで買える安いものでも効果ある？" >}}
十分効果があります。プチプラブランド（1,000〜2,000円）でも**毎日続けること**が何より重要です。
{{< /accordion >}}

{{< accordion title="ヒゲ剃り後の肌荒れにも使える？" >}}
シェービング後は**肌が傷ついている状態**なので、低刺激の化粧水＋乳液で必ず保湿を。
{{< /accordion >}}
