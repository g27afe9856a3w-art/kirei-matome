## 髪の悩み別おすすめカテゴリ

髪の悩みは人それぞれ。まず自分の悩みに合ったカテゴリを選ぶことが大切です。

### パサつき・ダメージ毛
カラーやパーマ、熱ダメージで髪が傷んでいる状態。**補修成分（ケラチン・CMC・アミノ酸）**が配合されたシャンプー＋濃密なトリートメントで集中補修を。

### 抜け毛・薄毛・頭皮トラブル
頭皮環境の悪化が主な原因。**スカルプケア系シャンプー＋育毛剤**の組み合わせが効果的です。洗浄力が強すぎるシャンプーは逆効果になる場合も。

### べたつき・脂性頭皮
皮脂分泌が過剰な状態。**アミノ酸系シャンプー**でやさしく洗いつつ頭皮の皮脂バランスを整えましょう。

### 敏感頭皮・かゆみ・フケ
刺激の少ない**アミノ酸系・低刺激シャンプー**がおすすめ。香料・防腐剤フリーの製品を選ぶと安心です。

### ボリュームダウン・ペタンコ髪
エイジングによる髪のコシ・ハリ不足。ボリュームアップ成分（コラーゲン・シルク）入りのアイテムで。

{{< info title="シャンプーの洗浄成分チェック方法" >}}
成分表示の先頭から数番目に書かれている成分が主な洗浄成分です。

- **アミノ酸系**（〜グルタミン酸Na、コカミドプロピルベタインなど）→ 低刺激・保湿力◎
- **石けん系**（石けん素地、脂肪酸Naなど）→ 洗浄力強め・環境にやさしい
- **高級アルコール系**（ラウレス硫酸Na、ラウリル硫酸Naなど）→ 洗浄力強・コスパ◎

敏感肌・ダメージ毛の方はアミノ酸系を選ぶと安心です。
{{< /info >}}

## ヘアケアの正しい手順

シャンプーの質を上げるだけでなく、**正しい手順で洗う**ことも同じくらい重要です。

{{< step num="1" title="ブラッシングで汚れを浮かせる" >}}
シャンプー前に乾いた状態でブラッシングを。頭皮の汚れ・スタイリング剤を浮かせることで洗浄効率がアップします。目の粗いコームやパドルブラシがおすすめ。
{{< /step >}}

{{< step num="2" title="予洗いで汚れの8割を落とす" >}}
38〜40℃のぬるめのお湯で1〜2分間しっかり予洗い。シャンプーの前の予洗いだけで汚れの約70〜80%が落ちると言われています。頭皮をマッサージしながら洗いましょう。
{{< /step >}}

{{< step num="3" title="シャンプーは手のひらで泡立ててから" >}}
シャンプーを直接頭皮につけるのはNG。手のひらで適量（500円玉大）を泡立ててから頭皮全体に広げます。爪を立てず、指の腹でやさしくマッサージするように洗うのがポイント。
{{< /step >}}

{{< step num="4" title="しっかりすすぐ（時間をかけて）" >}}
すすぎ残しはかゆみ・フケ・抜け毛の原因に。シャンプーの2〜3倍の時間をかけてしっかりすすぎましょう。耳の後ろ・えり足・後頭部は特に念入りに。
{{< /step >}}

{{< step num="5" title="トリートメントは毛先から" >}}
トリートメントは頭皮につけず、毛先から中間部分に集中してなじませます。2〜3分置いてから洗い流すと浸透率UP。
{{< /step >}}

{{< step num="6" title="タオルドライ＋ドライヤーで完全乾燥" >}}
濡れた髪はキューティクルが開いた状態で摩擦に弱いです。タオルでゴシゴシこすらず、押さえるように水分を取ります。その後ドライヤーで根元から乾かし、**半乾きのまま放置は厳禁**。雑菌・頭皮トラブルの原因になります。
{{< /step >}}

## よくある質問

{{< accordion title="シャンプーは毎日しないとダメ？" >}}
頭皮の皮脂分泌が多い方や汗をかく方は毎日シャンプーが基本です。ただし皮脂分泌が少ない乾燥肌の方は、1日おきにするか、ぬるま湯だけの「お湯洗い」を取り入れる方法も。ご自身の頭皮タイプに合わせましょう。
{{< /accordion >}}

{{< accordion title="シャンプーとトリートメントは同じブランドで揃えるべき？" >}}
同じブランドで揃えると処方の相性が良く効果を発揮しやすいですが、必須ではありません。ただしシャンプーとトリートメントで「ノンシリコン×シリコン入り」などの組み合わせは相性が良く、むしろ別ブランドを混ぜることも有効です。
{{< /accordion >}}

{{< accordion title="育毛剤はいつから使い始めるべき？" >}}
抜け毛が気になり始めたら早めのスタートがおすすめです。育毛剤は予防・改善のどちらにも有効で、特に「最近抜け毛が増えた気がする」「分け目が薄くなってきた」と感じ始めたタイミングが適切です。効果が出るまで最低3〜6ヶ月は継続してください。
{{< /accordion >}}

{{< accordion title="ドライヤーは髪に悪い？" >}}
正しく使えばドライヤーは髪を守ります。むしろ自然乾燥の方が問題です。半乾きの状態が長時間続くと雑菌が繁殖しやすく、頭皮トラブルの原因に。ドライヤーは熱風を同じ場所に当て続けず、15cm以上離して動かしながら使うのがポイントです。
{{< /accordion >}}
