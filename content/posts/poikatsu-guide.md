## 月1万円を達成する3つの戦略

{{< point title="稼ぐ人がやっている3つの方法" >}}
**戦略①：クレジットカード案件（1件5,000〜15,000円）**
ポイントサイト経由でカードを申し込むだけ。月1〜2件こなすだけで**1万円超えも可能**。

**戦略②：証券・FX口座開設（1件数千〜数万円）**
使っていない証券口座を開設して1回取引するだけ。高額案件が多い。

**戦略③：アンケート・ゲーム（コツコツ型）**
隙間時間で月2,000〜3,000円。少額だが積み重ねで安定収入に。
{{< /point >}}

## 注意点と安全に稼ぐコツ

{{< caution title="失敗しないための3つの注意点" >}}
1. **カード申し込みは6ヶ月に1〜2枚まで**：多すぎるとカード審査に影響する
2. **怪しいポイントサイトは使わない**：ハピタス・モッピーなど大手のみ利用
3. **ポイントの有効期限を管理する**：期限切れで失効するサイトもある
{{< /caution >}}

## よくある質問

{{< accordion title="会社員でもポイ活はできる？" >}}
できます。ポイ活はアルバイトや副業とは異なるため、副業禁止の会社員でも問題ないケースがほとんどです。ただし、年間20万円を超えると確定申告が必要になる場合があります。
{{< /accordion >}}

{{< accordion title="クレジットカードを作りすぎると信用情報に影響する？" >}}
短期間に多数のカードを申し込むと「多重申し込み」として審査に影響します。6ヶ月に1〜2枚のペースが安全です。
{{< /accordion >}}

{{< accordion title="家族で一緒にポイ活できる？" >}}
家族それぞれがポイントサイトに登録してカードを申し込めます。ただし同一カードに家族全員で申し込むことはできません（1人1枚が原則）。
{{< /accordion >}}
