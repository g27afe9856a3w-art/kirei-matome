## ホワイトニングの種類と違い

まず、ホワイトニングには大きく3種類あります。自分のライフスタイルや予算に合わせて選ぶことが大切です。

### 1. 歯科ホワイトニング（オフィスホワイトニング）

歯科医院で歯科医師・歯科衛生士が施術する方法です。**高濃度の薬剤を使用するため、効果が高く短期間で白さを実感**できます。

| 項目 | 内容 |
|------|------|
| 施術場所 | 歯科医院 |
| 料金相場 | 10,000〜30,000円（1回） |
| 効果の強さ | 高い |
| 施術時間 | 60〜90分 |
| 持続期間 | 6ヶ月〜1年 |

### 2. セルフホワイトニング

ホワイトニング専門サロンに行き、自分で機器を操作して行う方法です。**歯科医院より安くて手軽**に試せます。

| 項目 | 内容 |
|------|------|
| 施術場所 | 専門サロン |
| 料金相場 | 3,000〜8,000円（1回） |
| 効果の強さ | 中程度 |
| 施術時間 | 30〜60分 |
| 持続期間 | 1〜3ヶ月 |

### 3. ホームホワイトニング

歯科医院で作ったマウスピースと専用薬剤を使って、自宅でケアする方法です。**毎日少しずつ続けることで効果を出す**方法です。

| 項目 | 内容 |
|------|------|
| 施術場所 | 自宅 |
| 料金相場 | 10,000〜25,000円（キット） |
| 効果の強さ | 中〜高 |
| 施術時間 | 1〜2時間/日 |
| 持続期間 | 6ヶ月〜1年 |

{{< caution title="市販のホワイトニング商品に注意" >}}
市販のホワイトニング歯磨き粉やジェルは、日本では歯科医院で使うものよりも薬剤濃度が低く規制されています。劇的な効果は期待しにくいため、本格的な白さを求める方はクリニックでの施術がおすすめです。
{{< /caution >}}

## よくある質問

{{< accordion title="ホワイトニングは歯に悪影響はありませんか？" >}}
適切に行えば歯に大きなダメージはありません。ただし、施術後に一時的に知覚過敏が起こることがあります。歯科医院での施術であれば、歯の状態を確認しながら進めるため安心です。虫歯や歯周病がある場合は、先に治療を行う必要があります。
{{< /accordion >}}

{{< accordion title="ホワイトニングの効果はどのくらい続きますか？" >}}
種類によって異なりますが、歯科ホワイトニングで6ヶ月〜1年、セルフホワイトニングで1〜3ヶ月が目安です。食生活や習慣によって個人差があります。定期的にメンテナンスを行うことで、白さを長く維持できます。
{{< /accordion >}}

{{< accordion title="ホワイトニングはどのくらいで白くなりますか？" >}}
歯科オフィスホワイトニングでは1〜2回の施術で効果を実感できることが多いです。セルフホワイトニングは3〜5回程度通うと変化が見えてきます。元の歯の色や生活習慣によって個人差があります。
{{< /accordion >}}

{{< accordion title="被せ物や詰め物はホワイトニングできますか？" >}}
残念ながら、セラミックや樹脂などの被せ物・詰め物はホワイトニングの効果が出ません。ホワイトニングは天然歯にのみ効果があります。被せ物がある場合は、担当の歯科医師に相談してみましょう。
{{< /accordion >}}

{{< accordion title="セルフと歯科どちらがおすすめですか？" >}}
目的によります。「コスパよく気軽に試したい」ならセルフホワイトニング、「確実に白くしたい・歯の健康も気になる」なら歯科ホワイトニングがおすすめです。まずは無料カウンセリングを受けてみるのも良いでしょう。
{{< /accordion >}}
