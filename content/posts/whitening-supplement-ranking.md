## 塗る日焼け止めとの「二重防御」が最強

{{< point title="内外ダブルUVケアのやり方" >}}
| タイミング | やること |
|-----------|---------|
| 朝（食後） | 美白サプリを飲む |
| 外出30分前 | SPF30以上の日焼け止めを塗る |
| 2〜3時間後 | 日焼け止めを塗り直す |
| 夜（就寝前） | ビタミンC・コラーゲンサプリを飲む |

**サプリ単体ではなく、外側のケアとセット**で使うのが最も効果的です。
{{< /point >}}

## いつから飲み始めるべき？

{{< memo title="日焼けシーズン前の3ヶ月前がベスト" >}}
**おすすめのタイミング**：**3月から飲み始める**

紫外線が強くなる5〜8月に向けて、3月から体内でメラニン対策ができている状態を作るのが理想的です。「日焼けしてから飲む」では遅いので予防が大切！
{{< /memo >}}

## よくある質問

{{< accordion title="飲む日焼け止めは日焼け止めクリームの代わりになる？" >}}
なりません。あくまで補助的な役割です。外出時は必ずSPF30以上の日焼け止めを塗りましょう。
{{< /accordion >}}

{{< accordion title="妊娠中・授乳中でも飲める？" >}}
医薬品（トランシーノ・シナールL等）は医師に相談してから使用してください。サプリメントについても念のため医師に確認することをおすすめします。
{{< /accordion >}}

{{< accordion title="飲み合わせで注意することは？" >}}
鉄分サプリとビタミンCを同時に摂ると鉄の吸収が良くなりすぎる場合があります。複数サプリを飲む場合は成分の重複に注意しましょう。
{{< /accordion >}}
