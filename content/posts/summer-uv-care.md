## 4月から始めるべき理由｜紫外線は春から強い

{{< caution title="見落とされがちな春の紫外線" >}}
気象庁のUVインデックスデータによれば、**4月下旬の紫外線量は9月並み、5月には7月・8月とほぼ同等**になります。「夏だけUVケア」では遅すぎるのです。
{{< /caution >}}

紫外線にはUV-AとUV-Bの2種類があります。

| 種類 | 特徴 | 対策指標 |
|---|---|---|
| **UV-A** | 肌の奥まで届き、シワ・たるみの原因 | **PA（+〜++++）** |
| **UV-B** | 肌表面を焼き、シミ・そばかすの原因 | **SPF（1〜50+）** |

UV-Aは窓ガラスを通過するため、**室内にいても油断禁物**です。

## SPF/PAの正しい選び方｜シーン別ガイド

{{< point title="迷ったらこれ" >}}
- **日常・オフィス**: SPF20〜30 / PA++ 程度で十分
- **屋外レジャー・海**: SPF50+ / PA++++ 必須
- **敏感肌・子供**: ノンケミカル（紫外線散乱剤のみ）を選ぶ
{{< /point >}}

高SPFほど肌負担が増えるため、シーンに合った最小限のSPFを選ぶのが鉄則です。

## 塗る×飲む×グッズの最強コンボUVケア

「塗るだけ」では限界があります。2026年の最先端は**三層防御**です。

{{< step num="1" title="塗る日焼け止め｜基本のバリア" >}}
朝のスキンケア後に必ず塗布。2〜3時間おきの塗り直しが理想です。
{{< /step >}}

{{< step num="2" title="飲むUVサプリ｜内側からカバー" >}}
シダ植物由来のフォルノクリン、アスタキサンチン、リコピンなどが配合されたサプリが主流。詳しくは[飲む日焼け止め・美白サプリランキング]({{< ref "posts/whitening-supplement-ranking.md" >}})で解説しています。
{{< /step >}}

{{< step num="3" title="UVグッズ｜物理ブロック" >}}
日傘・帽子・アームカバー・UVカットサングラスで物理的に遮断。UPF50+の日傘は必須アイテムです。
{{< /step >}}

{{< info title="知ってた？" >}}
眼から入る紫外線もシミの原因になります。UVカットのサングラスは"美白ケア"としても有効です。
{{< /info >}}

## 日焼けしてしまった後のアフターケア

{{< caution title="日焼け直後の48時間が勝負" >}}
紫外線を浴びた肌は一時的な「軽いやけど」状態。メラニンが定着する前に素早くケアすることでシミになるリスクを減らせます。
{{< /caution >}}

1. **冷やす**: 濡らしたタオルや保冷剤で火照りを鎮める
2. **保湿**: 大容量の化粧水でたっぷり水分補給
3. **ビタミンC**: サプリや美容液で内外からメラニン対策
4. **睡眠**: ターンオーバーを促進するため早めに就寝

特にビタミンCは継続的な摂取がシミ予防に効果的。[ビタミンCサプリおすすめランキング]({{< ref "posts/vitamin-c-supplement-ranking.md" >}})も参考にしてください。
