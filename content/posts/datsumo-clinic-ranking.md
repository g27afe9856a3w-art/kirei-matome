## 脱毛クリニックの選び方3つのポイント

{{< point title="ポイント①：「全身」の定義を確認する" >}}
クリニックによって「全身」に**顔・VIOが含まれるかどうか**が異なります。含まれない場合、追加料金が発生するので必ず確認しましょう。
{{< /point >}}

{{< point title="ポイント②：通いやすい立地か確認する" >}}
脱毛は**5〜8回通う**必要があります。職場や自宅の近くに院があるかが長続きの鍵です。
{{< /point >}}

{{< caution title="注意：契約前に必ず無料カウンセリングへ" >}}
脱毛は数万円の契約になります。**必ず無料カウンセリングで料金・回数・効果を確認**してから契約しましょう。当日契約を急かすクリニックには注意が必要です。
{{< /caution >}}

## よくある質問

{{< accordion title="脱毛クリニックとサロンの違いは？" >}}
**クリニック**は医療レーザーを使うため永久脱毛が可能。**サロン**は光脱毛で減毛・抑毛が目的。完全に毛をなくしたいならクリニックがおすすめです。
{{< /accordion >}}

{{< accordion title="何回通えば効果が出る？" >}}
医療レーザーの場合、**5〜8回**で多くの方が満足できる効果を実感しています。部位や毛質によって異なります。
{{< /accordion >}}

{{< accordion title="痛みはどのくらい？" >}}
部位によりますが、**麻酔クリームを使えばほとんど痛くない**という方が多いです。VIOや顔はやや痛みを感じやすい傾向があります。
{{< /accordion >}}

{{< accordion title="学生でも申し込める？" >}}
18歳以上であれば申し込めます（未成年は保護者同意が必要な場合あり）。**学割があるクリニックも多い**ので、学生証を持参しましょう。
{{< /accordion >}}
