## AGAとは？症状と原因を知ろう

**AGA（Androgenetic Alopecia）**は、男性型脱毛症の医学的な名称です。日本人男性の約3人に1人が悩むと言われている非常に一般的な症状です。

### AGAの主な症状

- 頭頂部・生え際の毛が薄くなる
- 抜け毛の量が増える
- 毛が細くなる・短くなる
- ヘアラインが後退していく

### AGAの原因

AGAの主な原因は**男性ホルモン（テストステロン）**と**遺伝**の組み合わせです。

| 原因 | 詳細 |
|------|------|
| DHT（ジヒドロテストステロン） | テストステロンが変換されてできる物質。毛乳頭細胞に作用して毛周期を乱す |
| 遺伝 | 父方・母方どちらかにAGAがあると発症リスクが上がる |
| 生活習慣 | 睡眠不足・ストレス・食生活の乱れが促進要因になる |

{{< info title="AGAは自然には治らない" >}}
AGAは放置しても自然には回復しません。むしろ時間が経つほど毛根が萎縮し、治療が困難になります。「薄くなってきた」と感じたら、早めに専門クリニックへの相談をおすすめします。
{{< /info >}}

## なぜ早期治療が重要なのか

AGA治療は、**早く始めるほど効果が出やすく、費用も抑えられる**という大きなメリットがあります。

{{< step num="1" title="毛根が生きているうちに治療を始める" >}}
毛根細胞が生きている段階であれば、薬による治療で発毛を促せます。毛根が完全に死んでしまうと、植毛以外では対応できなくなります。
{{< /step >}}

{{< step num="2" title="薬の効果が出やすい" >}}
薄毛の進行が軽度な段階の方が、治療薬（フィナステリド・ミノキシジルなど）の効果が出やすいとされています。
{{< /step >}}

{{< step num="3" title="精神的な負担を減らせる" >}}
薄毛が目立つ前に治療を始めることで、コンプレックスによるストレスを軽減でき、仕事や私生活への悪影響を防げます。
{{< /step >}}

{{< caution title="市販の育毛剤では限界がある" >}}
市販の育毛剤は発毛促進効果を謳うものがありますが、AGAの根本的な原因（DHT）に作用する成分は含まれていません。AGA専用の処方薬（フィナステリド・デュタステリドなど）はクリニックでしか処方できません。
{{< /caution >}}

### 治療期間の目安

{{< memo title="AGA治療は継続が重要" >}}
AGA治療の効果は、一般的に治療開始から**3〜6ヶ月**で実感できることが多いです。ただし、個人差があり、1年以上かかるケースもあります。治療中断すると薄毛が再び進行するため、継続的な治療が重要です。
{{< /memo >}}

| 期間 | 変化の目安 |
|------|---------|
| 1〜3ヶ月 | 抜け毛が減少し始める |
| 3〜6ヶ月 | 毛の太さ・量に変化が現れ始める |
| 6〜12ヶ月 | 明らかな発毛効果を実感 |
| 1年〜 | 効果を維持しながら継続治療 |

## よくある質問

{{< accordion title="抜け毛が増えた気がするのですが、すぐに受診すべきですか？" >}}
1日に50〜100本程度の抜け毛は正常範囲ですが、それ以上に増えた・地肌が透けて見えてきたと感じたら、早めにAGA専門クリニックへの相談をおすすめします。AGAは早期治療が効果的です。無料カウンセリングから始めると気軽に相談できます。
{{< /accordion >}}

{{< accordion title="AGA治療薬の副作用はありますか？" >}}
フィナステリド・デュタステリドには性欲減退・勃起機能への影響などの副作用が報告されています。ただし、発現頻度は低く、服薬を中止すると通常回復します。クリニックで処方を受ける際に、医師から詳しい説明を受けることができます。
{{< /accordion >}}

{{< accordion title="AGA治療は何歳から受けられますか？" >}}
18歳以上であればAGA治療を受けられる場合がほとんどです。20代からAGA治療を始める方も増えています。早期に始めるほど効果が出やすいため、若い段階での相談も推奨されています。
{{< /accordion >}}

{{< accordion title="育毛剤とAGA治療薬の違いは何ですか？" >}}
市販の育毛剤は頭皮環境を整えるものが多く、AGAの根本原因（DHT）への作用はほとんどありません。一方、AGA治療薬（フィナステリド・デュタステリドなど）はDHTの生成を抑制し、薄毛の根本原因に働きかけます。医師の処方が必要です。
{{< /accordion >}}

{{< accordion title="オンライン診療だけでAGA治療はできますか？" >}}
はい、可能です。クリニックフォア・リゲインなど、完全オンラインで診療から薬の処方・郵送まで対応しているクリニックがあります。ただし、症状が重い場合や植毛を検討する場合は対面診療が必要なこともあります。
{{< /accordion >}}
